# 国际化 (I18n)

XSideUI.i18n 模块为应用提供多语言支持。这是一个自动化的国际化解决方案，基于Qt原生翻译机制，提供简洁的API和灵活的翻译体验。

## 导入

```python
from xsideui import tr, XI18N
```

## 核心组件

### 静态标记函数 tr(text)

```python
def tr(text: str) -> str:
    """
    仅用于静态标记，不执行实际翻译。
    作用：让 lupdate 能够提取到这个字符串，但在运行时它原样返回。
    """
    return text
```

**职责**：作为 lupdate 工具的"雷达信号"。
**用法**：所有需要翻译的常量字符串必须用 tr() 包裹。
**注意**：它在运行时不执行翻译（原样返回），仅用于静态扫描。

### 国际化管理器 I18nManager (单例 XI18N)

```python
class I18nManager(QObject):
    """
    国际化管理器（单例模式）

    职责：
    1. 管理和加载 .qm 翻译文件（包括组件库、Qt 标准库、用户自定义库）。
    2. 提供全局语言切换接口。
    3. 维护自定义翻译路径和上下文（Context）映射。
    4. 触发全局 UI 刷新事件。
    """
```

## 核心API

### XI18N.x_tr(text, context=None, args=None)

智能翻译函数，主要用于处理带占位符的动态翻译。

**参数：**

- `text`: 翻译键 (Key)，通常通过 tr() 传入
- `context`: 翻译上下文（可选）
- `args`: 占位符填充值列表，按顺序对应 %1, %2...

**返回值：** 翻译后的字符串

**主要用途：** 处理需要动态替换变量的翻译场景

**示例：**

```python
# 处理占位符翻译
raw_key = tr("User %1 has %2 messages.")
text = XI18N.x_tr(raw_key, args=['Tom', '15'])
label = XLabel(text=text)
```

### XI18N.add_custom_lang_path(dir_path, context="Global")

注册外部翻译目录。

**参数：**

- `dir_path`: 存放 .qm 文件的文件夹
- `context`: 翻译所属的上下文

**返回值：** 成功返回 True，失败返回 False

**示例：**

```python
XI18N.add_custom_lang_path("/path/to/translations", "MyCustomLib")
```

### XI18N.set_language(lang_code)

切换全局语言。

**参数：**

- `lang_code`: 语言代码，如 'zh_CN', 'en_US'

**返回值：** 成功返回 True，失败返回 False

**示例：**

```python
XI18N.set_language("zh_CN")
XI18N.set_language("en_US")
```

### XI18N.refresh_ui()

触发全局UI刷新，通知所有组件重新加载翻译。

**示例：**

```python
XI18N.refresh_ui()
```

## 标准开发工作流

### 1. 代码编写与标记

使用 tr() 标记需要翻译的字符串：

```python
from xsideui import tr

# 基础翻译 - 组件自动处理
btn = XButton(tr("Download"))

# 静态文本 - 组件自动处理
label = XLabel(tr("Welcome to My App"))

# 动态占位符 - 需要手动调用 x_tr()
raw_key = tr("User %1 has %2 messages.")
text = XI18N.x_tr(raw_key, args=['Tom', '15'])
dynamic_label = XLabel(text=text)
```

### 2. 提取词条 (lupdate)

运行命令扫描项目，生成或更新 .ts 翻译源文件：

```bash
# 扫描当前项目，生成 zh_CN.ts
pyside2-lupdate demo.py -ts ./custom_langs/zh_CN.ts
```

### 3. 人工翻译 (Qt Linguist)

使用 Qt Linguist 工具打开 .ts 文件：

- 找到对应的 Source Text
- 填写 Translation（注意保留 %1 等占位符）
- 点击"蓝色钩子"确认词条
- 点击发布生成 .qm 文件

### 4. 部署翻译文件

将生成的 .qm 文件放置在指定目录：

```
resources/translations/
├── zh_CN.qm
├── en_US.qm
└── ja_JP.qm
```

## 翻译模式

### 模式一：静态文本自动翻译

对于静态文本，XSideUI 组件会自动处理翻译：

```python
from xsideui import XButton, XLabel, tr

# 按钮自动翻译
button = XPushButton(tr("Download"))

# 标签自动翻译  
label = XLabel(tr("Welcome to My App"))

# 工具提示自动翻译
button.setToolTip(tr("Click to download file"))
```

### 模式二：动态占位符手动翻译

对于需要动态替换变量的文本，需要手动调用 `x_tr()`：

```python
from xsideui import XLabel, tr, XI18N

# 动态用户信息
raw_key = tr("User %1 has %2 messages.")
text = XI18N.x_tr(raw_key, args=['Tom', '15'])
label = XLabel(text=text)

# 动态任务状态
raw_key = tr("Task %1 is %2% complete.")
text = XI18N.x_tr(raw_key, args=['Backup', '75'])
status_label = XLabel(text=text)

# 动态时间显示
raw_key = tr("Last updated: %1")
text = XI18N.x_tr(raw_key, args=['2024-01-15 14:30'])
time_label = XLabel(text=text)
```

## 完整示例

```python
import sys
from PySide2.QtWidgets import QApplication, QWidget, QVBoxLayout
from xsideui import XButton, XLabel, tr, XI18N

class DemoWidget(QWidget):
    def __init__(self):
        super().__init__()
        layout = QVBoxLayout(self)



        # 静态文本 - 自动翻译
        welcome_label = XLabel(tr("Welcome to My App"))
        layout.addWidget(welcome_label)

        download_btn = XPushButton(tr("Download"))
        layout.addWidget(download_btn)

        # 动态占位符 - 手动调用 x_tr()
        raw_key = tr("User %1 has %2 messages.")
        text = XI18N.x_tr(raw_key, args=['Tom', '15'])
        user_label = XLabel(text=text)
        layout.addWidget(user_label)

        # 语言切换按钮
        lang_btn = XPushButton(tr("Switch to Chinese"))
        lang_btn.clicked.connect(lambda: XI18N.set_language("zh_CN"))
        layout.addWidget(lang_btn)

if __name__ == "__main__":
    app = QApplication(sys.argv)
    
   # 添加自定义翻译路径
   XI18N.add_custom_lang_path("./translations", "MyApp")

    # 设置默认语言
    XI18N.set_language("zh_CN")

    # 显示窗口
    demo = DemoWidget()
    demo.show()

    sys.exit(app.exec_())
```

## 翻译文件管理

### 默认翻译路径

组件库的翻译文件默认存放在：

```
src/xsideui/i18n/custom_langs/
├── zh_CN.qm
├── en_US.qm
└── ja_JP.qm
```

### 自定义翻译路径

支持添加多个自定义翻译路径：

```python
from xsideui import XI18N

# 添加多个自定义翻译路径
XI18N.add_custom_lang_path("./translations/module1", "Module1")
XI18N.add_custom_lang_path("./translations/module2", "Module2")
```

### Qt 官方翻译

自动加载 Qt 官方翻译，支持 Qt 内置组件的国际化：

```python
# Qt 原生对话框会自动翻译
from PySide2.QtWidgets import QFileDialog, QMessageBox

file_path, _ = QFileDialog.getOpenFileName(self, "Select File", "", "All Files (*)")
QMessageBox.question(self, "Question", "Do you want to save?",
                   QMessageBox.Yes | QMessageBox.No | QMessageBox.Cancel)
```

## 特性

- ✅ **智能翻译**：静态文本自动翻译，动态文本手动控制
- ✅ **占位符处理**：支持 %1, %2 等占位符动态替换
- ✅ **Qt集成**：自动加载Qt官方翻译
- ✅ **动态切换**：支持运行时语言切换
- ✅ **自定义路径**：支持多个自定义翻译目录
- ✅ **单例模式**：全局统一的翻译管理
- ✅ **自动刷新**：语言切换时自动更新UI

## 使用指南

### 何时使用自动翻译

**使用组件自动翻译（推荐）**：

- 静态按钮文本
- 固定标签文本
- 工具提示文本
- 菜单项文本

```python
# 自动翻译
button = XPushButton(tr("Download"))
label = XLabel(tr("Welcome"))
```

### 何时使用手动翻译

**手动调用 x_tr()**：

- 包含动态变量的文本
- 需要实时更新的状态信息
- 用户相关的个性化信息

```python
# 手动翻译（动态内容）
raw_key = tr("User %1 has %2 messages.")
text = XI18N.x_tr(raw_key, args=[username, message_count])
label = XLabel(text=text)
```

### 占位符使用规范

```python
# 正确使用
raw_key = tr("Task %1 completed in %2 seconds.")
text = XI18N.x_tr(raw_key, args=['Backup', '45'])

# 占位符从 %1 开始
raw_key = tr("File %1 of %2")
text = XI18N.x_tr(raw_key, args=['3', '10'])

# 支持多个占位符
raw_key = tr("User %1 from %2 has %3 items.")
text = XI18N.x_tr(raw_key, args=['Tom', 'Beijing', '5'])
```

## 注意事项

1. **翻译文件命名**：必须命名为 `{lang_code}.qm`，如 `zh_CN.qm`
2. **Context命名**：建议使用大驼峰命名法，如 `MyApp`、`Module1`
3. **占位符使用**：占位符从 `%1` 开始，不是 `%0`
4. **静态标记**：所有需要翻译的字符串必须用 `tr()` 包裹
5. **文件路径**：确保翻译文件路径正确且可访问
6. **区分使用场景**：静态文本用自动翻译，动态文本用手动翻译

## 故障排除

### 翻译不生效

1. 检查是否使用了 XSideUI 组件
2. 检查翻译文件路径是否正确
3. 确认翻译文件已正确编译
4. 验证语言代码是否正确

### 占位符不替换

1. 确认占位符格式为 `%1`, `%2`...
2. 检查是否正确调用了 `x_tr()` 函数
3. 验证 `args` 参数是否正确传递
4. 检查翻译文件中占位符是否保留

### 语言切换无效

1. 确认 `set_language()` 调用成功
2. 检查翻译文件是否存在
3. 验证是否使用了 XSideUI 组件

## 最佳实践

### 代码组织

```python
# 集中定义翻译键
class TranslationKeys:
    WELCOME = tr("Welcome to My App")
    DOWNLOAD = tr("Download")
    USER_STATUS = tr("User %1 has %2 messages.")

# 使用翻译键
button = XPushButton(TranslationKeys.DOWNLOAD)

# 动态翻译
raw_key = TranslationKeys.USER_STATUS
text = XI18N.x_tr(raw_key, args=['Tom', '15'])
label = XLabel(text=text)
```

### 性能优化

```python
# 避免在循环中重复调用 x_tr()
# 不推荐
for i in range(100):
    text = XI18N.x_tr(tr("Item %1"), args=[str(i)])
    label = XLabel(text=text)

# 推荐
raw_key = tr("Item %1")
for i in range(100):
    text = XI18N.x_tr(raw_key, args=[str(i)])
    label = XLabel(text=text)
```
